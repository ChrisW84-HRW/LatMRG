Detailed Description

template<typename Int>
class LatMRG::PrimitiveInt< Int >

This class deals with primitive roots and primitive elements modulo an integer.

Suppose that \(a\), \(e\), and \(p\) are integers, \(p\) is a prime number, and \(a\) is relatively prime with \(m=p^e\). The smallest positive integer \(\lambda(m,a)\) for which \(a^{\lambda(m,a)}= 1 \ \bmod\ m \) is called the order of \(a\) modulo \(m\). Any \(a\) which has the maximum possible order for a given \(m\) is called a primitive root modulo \(m\). This class provides functions to test for this property.

Detailed Description

This class constructs and handles lattice bases built from MRGs as in MRGLattice, but using using arbitrary lacunary indices.

that are regularly spaced by packets of the same size.

Constructor with modulus of congruence \(m\), order of the recurrence \(k\), multipliers \(A\), maximal dimension maxDim, and lattice type lattype.

Vector and matrix indices vary from 1 to maxDim. The length of the basis vectors is computed with norm. The bases are built using the lacunary indices lac. The basis is built for the lacunary indices lac. aa has to be a vector of k+1 components with a[i]= \(a_i\) for compatibility with other classes.

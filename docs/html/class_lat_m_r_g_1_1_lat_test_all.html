Detailed Description

template<typename Int, typename Dbl>
class LatMRG::LatTestAll< Int, Dbl >

This class is just an auxiliary class that allows launching the spectral, Beyer, or Palpha test on several different generators successively, each one associated with its own data file.

One can also launch these tests on all the data files (with extension ".dat") in a directory. The test and generator parameters in the data files must be as described in program LatMain (see page (FIXME: page#) of this guide). In fact, the LatMain program simply calls the methods of this class.

◆ doTest()

template<typename Int , typename Dbl >
int LatMRG::LatTestAll< Int, Dbl >::doTest ( const char *  datafile)

Reads the parameters of the test and of the generator in input text file datafile; then do the test.

The data file must always have the extension ".dat", but must be given as argument here without extension. For example, if the data file is named mrg.dat, then the method must be called as doTest("mrg"). Returns 0 if the test completed successfully; returns a negative integer if there was an error.

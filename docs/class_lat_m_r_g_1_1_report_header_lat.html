Detailed Description

template<typename Int, typename Dbl>
class LatMRG::ReportHeaderLat< Int, Dbl >

This class is an implementation of the ReportHeader abstract class for the programs lat*.

It prints the configuration of the test launched, the MRG or MWC components, and the combined MRG if applicable.

It can be the result of a combination of MRG components, a MWC transformed into a MRG, and so on.

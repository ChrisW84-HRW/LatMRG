Detailed Description

template<typename Int>
class LatMRG::IntPrimitivity< Int >

This class deals with primitive roots and primitive elements modulo an integer.

Let \(a\), \(e\) and \(p\) be integers, with \(p\) a prime number. Assume also that \(a\) and \(m=p^e\) are relatively prime. The smallest positive integer \(\lambda(m)\) for which \(a^{\lambda}= 1 \ \mod\ m \) is called the order of \(a\) modulo \(m\). Any \(a\) which has the maximum possible order for a given \(m\) is called a primitive root modulo \(m\). For the following important cases, the value of the order for given \(m\) is [22] :

\begin{align*} \lambda(2^e) & = 2^{e-2}, \qquad e > 2 \\ \lambda(p^e) & = p^{e-1}(p - 1), \qquad p > 2. \end{align*}
